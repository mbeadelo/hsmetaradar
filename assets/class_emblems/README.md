# Class Emblems

Place the class emblem images here with the following names:

- `deathknight.png` - Death Knight emblem
- `demonhunter.png` - Demon Hunter emblem
- `druid.png` - Druid emblem
- `hunter.png` - Hunter emblem
- `mage.png` - Mage emblem
- `paladin.png` - Paladin emblem
- `priest.png` - Priest emblem
- `rogue.png` - Rogue emblem
- `shaman.png` - Shaman emblem
- `warlock.png` - Warlock emblem
- `warrior.png` - Warrior emblem
- `neutral.png` - Neutral/Unknown class (fallback)

**Recommended size:** 64x64px or larger (square format)
**Format:** PNG with transparency preferred
